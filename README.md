#  Analyse and Visualise the Space Race

##  Overview
This project explores and visualizes historical data from the Space Race, covering all space missions since 1957. Using a dataset sourced from Next Spaceflight, the analysis focuses on uncovering trends, patterns, and insights related to mission success, costs, launch frequency, and country participation.

The goal is to transform raw data into meaningful insights through data cleaning, exploration, and visualization techniques.

---

## Objectives
- Analyze trends in space missions over time  
- Identify the most active countries and organizations  
- Evaluate mission success and failure rates  
- Study how mission costs have evolved  
- Discover seasonal patterns in launches  
- Build compelling visualizations to communicate insights  

---

##  Key Questions Explored
- Who launched the most missions in a given year?  
- How has the cost of space missions changed over time?  
- Which months have the highest number of launches?  
- Have missions become safer over time?  
- Which organizations dominate the space industry?
